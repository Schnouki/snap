Changelog

3.1.9 (2016/06/26)

  • Caches pages to prevent requesting the same page multiple times.
  • Pre-fetches images of new pages.
  • Supports optional session-wide page caches.
  • Shows real progress bar based on Content-Length header.
  • Supports ES6 modules build.
  • Fixes a bug where a same link is added to history API multiple times.
  • Adds an off method.
  • Makes preload method public.
  • Loads scripts synchronously (#85).

3.1.0 (2014/12/25)

  • Handle links added with JavaScript.
  • Allow to alter pages on receive.
  • Bug fixes.

See commits on GitHub.

3.0.1 (2014/03/21)

  • Bug fix for non-mobile touch browsers, where previously snap would throw an error.

See commits on GitHub.

3.0 (2014/03/08)

  • Preloads on “touchstart”, for mobile devices.
  • Progress bar, so the user sees when a page change happens.
  • New events.
  • Asset change detection.
  • Ability to blacklist or whitelist all links in a parent element.
  • Available via Bower: bower install instantclick
  • Bug fixes.

If you’d like to share the news or get more detailed informations, there’s an announcement page: snap 3.0 is out.

See commits on GitHub.

2.1 (2014/02/09)

  • Blacklisting is now the default (whitelisting by default was confusing).
  • Bug fix: HTML entities would show up in the title on page change.

See commits on GitHub.

2.0 (2014/02/08)

  • Option to set a delay before preloading starts.
  • Option to preload on mousedown.
  • Prevent the “preload, display, re-preload” situation by not preloading if a page has been displayed in the last 100 ms.
  • Set body’s outerHTML instead of innerHTML on page change.
  • Bug fix: Scroll correctly when using the browser’s back or forward button.

See commits on GitHub.

1.1 (2014/01/16)

  • Whitelist mode.
  • snap.on.
  • snap.supported.
  • Redirect if we’ve preloaded non-HTML content.
  • Bug fix: <a> elements can now contain child elements.
  • Bug fix on the browser’s back/forward button.

See commits on GitHub.

1.0 (2014/01/08)

Initial release.

See commits on GitHub.
